# Class Board Planner (Static HTML)

Простой планировщик оформления доски/стены в одном файле `index.html`.

## Как запустить

1. Откройте файл `index.html` в любом браузере.

## Что делает

- Принимает размеры стены (ширина/высота)
- Учитывает поля и нахлест для листов A4
- Считает точное количество листов
- Показывает визуальную раскладку плиток
- Предлагает 3 варианта композиции
