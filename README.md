# Квиз: таблица умножения и деления

Простое веб-приложение в одном файле `index.html` для тренировки умножения и деления.

## Запуск

1. Откройте `index.html` в браузере.

## Возможности

- 3 режима: только умножение, только деление, смешанный.
- Выбор максимального числа (до 5, 10, 12, 15, 20).
- Выбор количества вопросов (10, 15, 20, 30).
- Таймер прохождения квиза.
- Мгновенная проверка ответа.
- Финальный результат: правильные ответы, точность, время, оценка.
