# Class Board Generator MVP

MVP веб-приложения для учителей: ввод размеров стены, выбор темы/мероприятия, предложение композиции и автоматический расчет раскладки по листам A4.

## Запуск

1. Установить зависимости:

```bash
npm install
```

2. Запустить dev-сервер:

```bash
npm run dev
```

3. Открыть `http://localhost:3000`.

## Что уже работает

- Ввод параметров стены и оформления
- 3 композиционных варианта
- Расчет количества листов A4 (`cols x rows`)
- Визуальная схема раскладки плиток с нумерацией

## Принцип работы

Приложение ничего не печатает и не экспортирует файлы.
Пользователь видит итоговую раскладку и точное количество листов, которые нужно распечатать самостоятельно.
